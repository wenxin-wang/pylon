* Aider

** Github Copilot 支持

同事们说用 copilot.el 登录下就能获得 token，然后按 [[https://aider.chat/docs/llms/github.html#where-do-i-get-the-token][官方文档]] 用 OpenAI API 的方式去访问。实际这样用的话会遇到
API 请求被拒绝的情况。

实际 aider 用的 litellm 是支持 copilot 的，用 [[https://www.reddit.com/r/ChatGPTCoding/comments/1lk2mvv/aider_anyone_have_success_with_gh_copilot_oauth/][这里]] 和 [[https://github.com/Aider-AI/aider/issues/2227#issuecomment-3141551921][这里]] 推荐的配置直接就能用了，它自己会去发起申请 token
的流程。

** gpt.el

gpt.el 的 chat buffer 本身是 markdown/org-mode 格式的。https://github.com/jwiegley/ob-gptel 给 org-mode 增加了
src block 的支持。感觉都很有用。

#+begin_src gptel
What is the capital of France?
#+end_src

#+RESULTS:
: The capital of France is Paris.
